**Overview**
Smart Bite is a Power BI dashboard project that analyzes packaged food products to help users make informed purchasing decisions. The dashboard provides insights into nutritional content, product categories, ingredient composition, and overall food quality through interactive visualizations.
**Objectives**
-Analyze packaged food products using nutritional data.
-Compare products across different categories.
-Identify healthier food choices.
-Present insights through an interactive dashboard.
**Features**
-Interactive filters and slicers
-Category-wise analysis
-Nutritional comparison
-Product-level insights
-Easy-to-understand visualizations
**Tools & Technologies**
Power BI Desktop
Microsoft Excel
